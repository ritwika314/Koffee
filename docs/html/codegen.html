codegen.codegen(input_ast, symtab, tabs=0, wnum=0, stages=False)¶

The main code generation function. It takes as input an AST, and returns its corresponding java code. It is called recursively on the branches of the syntax tree.

Args:
input_ast (ast): if its not an AST, then return nothing, else generate code. tabs (int): indentation for generated java program.
Returns:
generated_code (str): java code as a string.
